# Projeto: Leado Contrato

## Estrutura do Projeto
O projeto está organizado da seguinte forma:
- **src/**: Código fonte da aplicação.
- **public/**: Arquivos estáticos que são servidos pela aplicação.
- **tests/**: Testes automatizados.

## URLs
As seguintes URLs estão disponíveis para interação com a aplicação:
- **Homepage**: `http://localhost:3000`  
- **API**: `http://localhost:3000/api`

## Produtos
1. **Implementação Express**: Framework usado para criar aplicações web e APIs.
2. **Atendimento Magnético Playbook**: Um guia para utilizar o Atendimento Magnético.
3. **Do Lead ao Contrato**: Documento que delineia o processo do lead até a formalização do contrato.
